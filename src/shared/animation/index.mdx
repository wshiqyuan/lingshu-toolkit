# animation

>
> shadcn any version

## install

```bash title="npm"
npm i @cmtlyt/lingshu-toolkit
```

```bash title="shadcn"
npx shadcn@latest add https://cmtlyt.github.io/lingshu-toolkit/r/sharedAnimation.json
```

## usage

```ts
import { animation } from '@cmtlyt/lingshu-toolkit/shared'
// or
import { animation } from '@cmtlyt/lingshu-toolkit/shared/animation'
```
